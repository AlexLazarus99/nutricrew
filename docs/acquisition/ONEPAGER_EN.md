## What it is

**NutriCrew** is a **team nutrition game inside Telegram**: snap food → AI calories → crew points → weekly leagues and Telegram Stars prizes.

Not another solo calorie app — **social accountability + gamification** native to group chats.

## Problem → solution

| Pain | NutriCrew |
|------|-----------|
| Solo trackers churn | Team pressure & leaderboards |
| App install friction | Telegram Mini App, zero install |
| Boring food logging | NutriBird, quests, streaks, XP |
