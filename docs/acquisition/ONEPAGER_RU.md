## Что это

**NutriCrew** — социальная игра про питание внутри **Telegram**: фото еды → AI-калории → очки команде → недельные лиги и призы (Stars).

Не «ещё один калькулятор», а **Duolingo для питания в групповом чате**.

## Проблема → решение

| Боль | Решение NutriCrew |
|------|------------------|
| 90% churn в solo-трекерах | Командное давление и лидерборды |
| Сложно установить приложение | Mini App в Telegram, 0 install |
| Скучно вести дневник | NutriBird, квесты, серии, XP |

## Что уже live

- Telegram **бот + Mini App** (14 языков UI)
- **AI vision** (OpenAI / Claude / Gemini) — разбор фото и текста
- **Команды**, чат, дуэли, квесты, battle pass
- **Монетизация:** LiteCrew, Pro, Telegram Stars, Tribute, рефералка, B2B-лендинг
- **Корп. задел:** organizations, org dashboard
- **Инфра:** Node.js, PostgreSQL (Neon), Prisma, S3, Render + Vercel

## Для кого покупка имеет смысл

1. **Нутришн-конкурент** (KAVA и др.) — готовый социальный слой в TG  
2. **Telegram-студия** — white-label wellness mini app  
3. **Фитнес-сеть / HR** — корпоративные челленджи под брендом  
4. **Acqui-hire** — команда + код за 4–8 недель интеграции
