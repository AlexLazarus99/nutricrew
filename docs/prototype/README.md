# NutriCrew UI Prototype

Interactive click-through prototype — **no backend or Telegram required**.

## Open

Double-click or serve locally:

```bash
# from project root
npx --yes serve docs/prototype -p 5199
```

Then open http://localhost:5199

Or open `index.html` directly in Chrome/Edge.

## Features

- All 5 screens: Home, Log meal, Team, Rank, Prizes
- Onboarding flow (no team) via toggle
- EN / RU language switch
- Realistic mock data (Protein Squad, weekly battle)
- Phone frame layout (390×844)

## For pitch

Use fullscreen (F11) or share screen during demo. Toggle **EN/RU** in header.
