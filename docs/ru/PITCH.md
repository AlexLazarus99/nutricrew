# NutriCrew

**Командные битвы по правильному питанию прямо в Telegram.**  
Фото → ИИ → очки команде → недельные войны с реальными призами.

---

## Проблема

Трекеры калорий — **одиночные, скучные, их бросают**. MyFitnessPal и KAVA считают еду, но **~90% уходят**, потому что мотивация без команды не держится. Корпоративный ЗОЖ растёт, а привычка не закрепляется в групповых чатах, где люди живут каждый день.

## Решение

**NutriCrew** = AI-скан еды + **лиги как в Duolingo** + **распространение через Telegram**.

| Шаг | Опыт |
|-----|------|
| 📸 Фото еды | ИИ → калории за ~3 сек |
| ⚡ Очки | Личная серия + множитель команды |
| 🏆 Победа недели | Рейтинг, призы Stars, статус |

Привычка через **социальное давление**: пропустил день — команда теряет бонус.

## Почему сейчас

- ✅ **AI-скан еды работает** — KAVA уже доказал петлю  
- ✅ **Telegram Mini Apps** — 900M+ пользователей, без установки, вирусность в чатах  
- ✅ **Командные игры** — Duolingo Leagues, Wordle, Strava  
- ✅ **Пустая монетизация** — в TG никто не монетизирует nutrition через **социальную модель**

## Монетизация

| Поток | Механика |
|-------|----------|
| **Призовые фонды** | Команда пополняет недельный банк **Telegram Stars** → 80% победителям |
| **Team Premium** | Бейдж и статус (~99 ⭐ / мес. на команду) |
| **B2B (далее)** | Корпоративные подписки, OKR по командным челленджам |

**Формула:** AI-скан + социальный retention + TG + Stars = **первая социальная nutrition-платформа в Telegram**.
